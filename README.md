# Gruvbox Theme for end-4 dots

A complete, muted Gruvbox color scheme for end-4 dots-hyprland (illogical-impulse).

## Features

- Authentic muted Gruvbox colors from the original Dark palette
- Colorful bar icons: CPU (red), RAM (yellow), Swap (blue), Battery (pink)
- Muted aqua workspace indicators
- Subtle UI backgrounds
- Animated blue-to-aqua gradient window borders
- Readable cream tooltips with dark text
- Update-resistant design

## Prerequisites

- end-4 dots-hyprland (illogical-impulse version)
- Hyprland
- Quickshell
- matugen

## Installation
```bash
git clone https://github.com/YOUR_USERNAME/gruvbox-end4-theme.git
cd gruvbox-end4-theme
./install.sh
```

The installer will backup your existing configuration first.

## After end-4 Updates

If end-4 dots updates overwrite your theme:
```bash
cd gruvbox-end4-theme
./install.sh
```

## Uninstallation
```bash
./uninstall.sh
```

## What Gets Modified

- matugen color templates
- Quickshell Appearance defaults
- Bar resource colors
- Battery indicator
- Hyprland window borders
